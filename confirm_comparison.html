MIGRATION PLAN

Each raw dialog follows the same pattern — AlertDialog(title, content, actions: [CANCEL, CONFIRM]). Migrating to KsConfirmDialog.show() replaces 20+ lines per site with a single call: 

// Before: 20 lines of raw AlertDialog
showDialog<bool>(builder: (ctx) => AlertDialog(
  backgroundColor: ctx.ksc.primary800,
  title: Text("DELETE ITEM?"),
  content: Text("This will be permanently removed."),
  actions: [TextButton(...), TextButton(...)],
));

// After: 1 line of KsConfirmDialog
final confirmed = await KsConfirmDialog.show(
  context,
  title: 'DELETE ITEM?',
  message: 'This will be permanently removed.',
  confirmLabel: 'DELETE',
  isDanger: true,
);

Result: consistent icon + animation + button styling across all 27 confirmation dialogs. Saves ~100 lines of duplicated code.
