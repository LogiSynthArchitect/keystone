SNACKBAR / FEEDBACK — DESIGN EXPLORATION

Feedback Notifications

Four options for how the app communicates success, error, and info to the user.

A — Slide-Up Toast

Evolution of current KsSnackbar

✓ Price updated successfully ✕
⚡ Failed to save price. Try again. ✕
ⓘ Price must be greater than 0 ✕
AspectBehavior
EnterSpring slide-up from bottom (cubic-bezier(0.34, 1.56, 0.64, 1))
ExitSlide down + fade, 300ms
PositionFloating at bottom, 16px from edges
SuccessGreen left border ✓ auto-dismiss 3s
ErrorRed left border ✓ stays until dismissed
Info/WarningGold left border ✓ auto-dismiss 3s
StackingMultiple stack upward with gap
B — Top Banner (Below App Bar)

Error demand attention at the top

⚠ Failed to save price. Check connection.
AspectBehavior
EnterSpring slide-down from top
ExitSlide up + fade, 300ms
PositionBelow app bar, pushes content down
SuccessGreen tinted ✓ auto-dismiss 4s
ErrorRed tinted ✓ stays until dismissed
WarningGold tinted ✓ auto-dismiss 4s
StackingOnly one banner at a time (replaces)
C — Floating Pill

Minimal, non-intrusive confirmation

✓ Price updated
AspectBehavior
EnterScale pop-in + fade
ExitScale down + fade, 250ms
PositionCentered at bottom, above content
SuccessGreen checkmark ✓ auto-dismiss 2s
ErrorRed icon — better suited for toast/banner
InteractionNone required — pure confirmation
COMPARISON — When To Use Which
ScenarioRecommendedWhy
✅ Price saved successfullyA — Toast or C — PillBrief confirmation, no action needed
❌ Save failed (network error)A — Toast (error variant)Stays visible, user must acknowledge
⚠️ Validation warningB — BannerInline, can't be missed, doesn't overlap content
💬 Info messageC — PillMinimal interruption
🔋 Offline / reconnectB — BannerKsOfflineBanner already does this pattern
MY RECOMMENDATION
Use B (Top Banner) for errors so they demand attention and don't overlap bottom navigation.
Use C (Floating Pill) for success confirmations — minimal, clean, auto-dismisses.
Scrap the current flat snackbar entirely. Both new options use spring animations consistent with NoirLuxe feel.
