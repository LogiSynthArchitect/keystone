Changes Summary — All Using Existing Data
① Summary Strip
Compact chip row below header showing counts of items, expenses, photos, notes, links. Instant overview without scrolling.
② Gross Profit Calculator
Revenue (final charge) minus cost of goods (parts + hardware costs + expenses). Shows margin visually with a progress bar.
③ Unified "Items Used"
Merge Parts + Hardware into a single section matching the add/edit job pattern. Color-coded dots (gold = hardware, blue = parts). Shows running total.
④ Customer Quick Actions
Adds tap-to-call, WhatsApp, and navigate buttons below the customer card. All from existing phone/location data.
⑤ Expense Running Total
Sum at the bottom of the expenses list. Same data, just visible.
⑥ Photo Type Badges
Show PHOTO/VIDEO/AUDIO badge on each media card matching the media drawer pattern.
Bottom line: Zero new API calls, zero new entities, zero new screens. All improvements use data already loaded by jobDetailProvider, jobPartsProvider, jobHardwareProvider, etc.
