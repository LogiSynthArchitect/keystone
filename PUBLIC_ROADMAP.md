# Keystone Public Roadmap

**Status:** V1 shipped and live with 2 pilot users. V2 in planning.

---

## V1 — Foundation (✓ Complete)

**Core Features:**
- ✓ Job logging with offline sync
- ✓ Customer history and deduplication
- ✓ Personal knowledge base (notes)
- ✓ WhatsApp one-tap follow-ups
- ✓ Public web profile for customer sharing
- ✓ Automatic background sync

**Why V1 was focused:** One problem, solved well. Locksmiths needed a way to not lose customer data. We built that.

---

## V2 — Growth (Planned Q2 2026)

**Wider Distribution:**
- Google Play Store launch (easier installation for users)
- Improved onboarding flow

**User Verification:**
- SMS-based identity verification (replacing test codes)
- Multi-device support (same account on multiple phones)

**Insights:**
- Basic analytics dashboard (how many jobs logged, busiest days)
- Job performance tracking (turnaround time, customer patterns)

**Admin Tools:**
- Dedicated UI for managing user roles and permissions
- Job correction workflow (users request fixes, admins approve)

**Accessibility:**
- Multi-language support (Twi, Ga, other local languages)
- Offline-first continued (even faster on slow networks)

---

## Beyond V2

- **Scale to 100+ technicians** — Proven product-market fit
- **Advanced analytics** — Revenue insights, customer lifetime value
- **API for partners** — Other tools can integrate with Keystone data
- **Community features** — Technicians share tips, best practices

---

## Philosophy

**Build narrow, deep, and with real users.** Each version solves the actual problem in front of us, not imagined future problems.

V1 proved we can build something locksmiths want. V2 proves we can scale it. Beyond that: we decide based on what real users ask for.
