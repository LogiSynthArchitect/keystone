HOW IT WORKS

Separate lists _beforePhotos: List<XFile> and _afterPhotos: List<XFile> stay separate. No MediaItem or MediaTag needed. The list IS the tag.
Tab = active list BEFORE tab shows _beforePhotos. AFTER tab shows _afterPhotos. Toggle tabs to switch. Upload adds to the active list.
Each tab = own ADD MEDIA Source picker appears in the active tab. Cards + shimmer in the active grid. Total count updates in header.
Maximize drawer height Use isScrollControlled: true with higher than default height so the grid has room. 2-col grid fills the space. Tall drawer = better photo browsing.
