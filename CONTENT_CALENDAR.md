## March 2026

### Post 1: Keystone Inception

**Status:** Ready to post (2026-03-21)

**Post Text:**
```
We built Keystone for locksmiths in Accra who were losing customer
data when their phones crashed.

The problem: A phone breaks → all customer info gone → can't follow up.
Jeremie (our pilot user) was losing jobs to forgotten customer details.

The solution: Offline-first mobile app + automatic sync.

How it works:
1️⃣ Write to local storage immediately (instant feedback)
2️⃣ User sees "saved" — no waiting for internet
3️⃣ Sync to server in background
4️⃣ User never waits for connectivity

Result: Jobs logged in <2 seconds. Synced in 10s on next connection.

Tech stack: Flutter + Supabase + Hive + Vercel
Built in 6 weeks with 2 pilot users.

This is the journey of shipping in public. Real problem. Real users.
Real bugs. Real fixes.

Follow along as we build V1 → V2 → scale.

GitHub: https://github.com/[your-repo]
```
