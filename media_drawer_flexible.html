FLEXIBLE DATA MODEL

/// A labeled group of media files for the media drawer. class MediaGroup { final String label; // "BEFORE PHOTOS" / "JOB PHOTOS" final List<XFile> files; // mutable list, drawer edits directly final String? emptyHint; // "Take a photo of the site first" const MediaGroup({ required this.label, required this.files, this.emptyHint, }); }
MediaGroup data class New file job_step_types.dart or inline. Stores label + files + optional empty hint. Caller passes a list.
_buildMediaGridSection Replaces _buildPhotoGroup. Takes a MediaGroup. Renders section header + 2-col grid + ADD MEDIA button + source picker.
_buildMediaCard Single card. Props: XFile, onRemove, isLoading. Inspects file extension for video/audio overlay.
_showMediaSourceSheet Bottom sheet: Camera / Gallery / Video / Audio. Returns the chosen source. Caller invokes ImagePicker after.
Shimmer loading While ImagePicker is processing, add a loading: true card to the grid. Animated shimmer via AnimationController.
Updated _showPhotosDrawer Now accepts List<MediaGroup> instead of hardcoded BEFORE/AFTER. Passes through from caller. _buildStep6 defines the groups.

USAGE EXAMPLES

// In _buildStep6 (Extras step): _showPhotosDrawer(groups: [ MediaGroup(label: "JOB PHOTOS", files: _jobPhotos, emptyHint: "Take a photo of the work done"), ]); // Or for before/after: _showPhotosDrawer(groups: [ MediaGroup(label: "BEFORE PHOTOS", files: _beforePhotos, emptyHint: "Take a photo before starting"), MediaGroup(label: "AFTER PHOTOS", files: _afterPhotos, emptyHint: "Take a photo after completing"), ]);
